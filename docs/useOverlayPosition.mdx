Handles positioning overlays like popovers and menus relative to a trigger element, and updating the position when the window resizes.

### Install

<Install p={"@react-native-aria/overlays"} />

### Import

```js
import { useOverlayPosition } from "@react-native-aria/overlays";
```

### API

```js
useOverlayPosition(props: AriaPositionProps): PositionAria
```

#### AriaPositionProps

Props for the [overlay position](https://react-spectrum.adobe.com/react-aria/useOverlayPosition.html).

#### PositionAria

[Position info](https://react-spectrum.adobe.com/react-aria/useOverlayPosition.html)
